# 研究紹介ファイル置き場

このフォルダには、Pull Request 演習で使う研究紹介ファイルを置きます。

各自が自分用の Markdown ファイルを1つ作成します。

ファイル名の例:

```text
yamada.md
sato.md
student01.md
```

公開できない研究内容、個人情報、未公開データ、認証情報は書かないでください。迷う場合は、架空の研究テーマを書いてもかまいません。

## 記入例

```markdown
# 山田の研究紹介

## 研究テーマ

水質データを使った季節変化の分析

## 研究の目的

川の水質が季節によってどのように変化するかを調べます。

## 使うデータ

- 月ごとの水温
- 月ごとの pH
- 月ごとの観測地点

## まだ分からないこと

- 雨が多い時期に値がどのくらい変わるか
```
